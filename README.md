# bahan-ajar-laravel

# Mulai Project

1. Buat project
composer create-project laravel/laravel crud-user

2. Jalankan project
php artisan serve

3. Ganti settingan .env untuk konfigurasi DB
4. Jalankan migrasi database
php artisan migrate
5. Buat model, migration, dan controller
php artisan make:model User -mcr
